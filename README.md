# Minha Evolução

PWA pessoal para acompanhamento de peso, cintura, IMC, percentual de gordura corporal, metas e planejamento mensal.

## Uso local

```powershell
node dev-server.mjs
```

Depois acesse:

```txt
http://127.0.0.1:4173
```

## Publicação

O app é estático e pode ser publicado diretamente no GitHub Pages a partir da branch `main`.
